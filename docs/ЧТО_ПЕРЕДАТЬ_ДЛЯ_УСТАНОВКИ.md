# Какие файлы передавать для установки на разных ОС

## Краткая сводка

| ОС / режим | Что передать | Документация |
|------------|--------------|--------------|
| **Windows (с GUI)** | `dist/Kvant Server Setup 1.0.0.exe` | РУКОВОДСТВО.md |
| **Linux (с GUI)** | `dist/linux-unpacked/` или AppImage (если собран) | РУКОВОДСТВО.md |
| **Linux headless** | `dist-headless/api-deployer-headless-1.0.0.tar.gz` | РУКОВОДСТВО.md, HEADLESS_FULL_MANUAL.md |

---

## 1. Windows (с графическим интерфейсом)

**Файл:** `dist/Kvant Server Setup 1.0.0.exe`

- Один установочный файл
- Пользователь запускает, выбирает папку, устанавливает
- Требуется: MySQL, Node.js не нужен (всё встроено)

**Документация:** `docs/РУКОВОДСТВО.md` или `files/docs/РУКОВОДСТВО.md`

---

## 2. Linux (с графическим интерфейсом)

**Папка:** `dist/linux-unpacked/` — распакованное приложение

Или **AppImage** (если собран): файл `*.AppImage` в `dist/`

- Для AppImage: сделать исполняемым (`chmod +x`) и запускать
- Для linux-unpacked: запускать `./kvant-server` (или имя из productName)

**Требования:** MySQL, библиотеки для Electron (обычно есть в современных дистрибутивах)

**Документация:** `docs/РУКОВОДСТВО.md`

---

## 3. Headless (без GUI) — Linux / Windows

**Файл:** `dist-headless/api-deployer-headless-1.0.0.tar.gz`

**Установка на сервере:**
```bash
tar -xzf api-deployer-headless-1.0.0.tar.gz
cd api-deployer-headless-1.0.0
npm ci --production
node scripts/setup-cli.js    # настройка
node server.js               # запуск
```

**Требования:** Node.js 18+, MySQL

**Документация:** 
- `docs/РУКОВОДСТВО.md` — общее руководство
- `files/docs/HEADLESS_FULL_MANUAL.md` — подробная инструкция по headless
- `files/docs/DEBIAN_FIRST_TIME.md` — первый запуск на Debian

---

## Минимальный набор для передачи

### Вариант A: Всё в одном архиве

Создайте папку `Kvant-Server-Release-1.0.0` и положите:

```
Kvant-Server-Release-1.0.0/
├── Windows/
│   └── Kvant Server Setup 1.0.0.exe
├── Linux-GUI/
│   └── (содержимое dist/linux-unpacked/ или AppImage)
├── Headless/
│   └── api-deployer-headless-1.0.0.tar.gz
└── docs/
    ├── РУКОВОДСТВО.md
    ├── HEADLESS_FULL_MANUAL.md      (если есть)
    └── DEBIAN_FIRST_TIME.md         (если есть)
```

Заархивируйте и передайте один файл.

### Вариант B: По запросу

- **Только Windows** → `Kvant Server Setup 1.0.0.exe` + РУКОВОДСТВО.md
- **Только Linux GUI** → папка linux-unpacked или AppImage + РУКОВОДСТВО.md  
- **Только headless** → `api-deployer-headless-1.0.0.tar.gz` + РУКОВОДСТВО.md + HEADLESS_FULL_MANUAL.md

---

## Что можно не передавать

- `*.blockmap` — служебные файлы для обновлений
- `*.7z` — промежуточный архив
- `builder-debug.yml`, `builder-effective-config.yaml` — отладочные файлы сборки
- Папки `win-unpacked`, `__appImage-x64` — если передаёте готовый установщик/архив
